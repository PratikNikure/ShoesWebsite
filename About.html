PURCHASES MADE THROUGH REEBOK UNLOCKED

1. Sign In to your Reebok UNLOCKED account
2. Go to your Order History
3. View the order you want to return
4. Click Return Item(s)
5. Check the item(s) you want to return and let us know why
6. Confirm your return
7. Follow the return shipment instructions below

PURCHASES MADE AS A GUEST

1. Go to our Order Tracker
2. Enter your order number (i.e. RB000000000) and email address, then click Track Order to view your order
3. Click Return Item(s)
4. Check the item(s) you want to return and let us know why
5. Confirm your return
6. Follow the return shipment instructions below

RETURN SHIPMENT INSTRUCTIONS

1. Print your return label. No printer? Simply save the FedEx QR code or return label on your mobile device to show during drop off
2. Pack the item(s) you want to return in the original box, or any available to you. If you have created more than one return label, please package the items separately, per return label. After your return label is created, it is only valid for 7 days before expiring.
3. Seal the box and place your return label over the original delivery label, or show the return FedEx QR code or return label during drop off
4. Bring your package to a return drop off location. More details on return drop off locations can be found at http://www.fedex.com/easyreturns
5. Refunds will be issued within 14 days of the return drop off
